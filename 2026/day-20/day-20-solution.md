# Day 20 – Log Analyzer & Report Generator

## Project Overview

Built a Bash-based Log Analyzer that automates log inspection and generates a summary report.

## Features

- Reads a log file from the command line
- Validates file existence
- Counts ERROR and Failed entries
- Detects CRITICAL events with line numbers
- Finds the top 5 most common error messages
- Generates a timestamped report
- Archives processed logs (optional)

## Commands Used

- grep
- awk (optional)
- cut
- sort
- uniq
- wc
- head
- mv
- date
